# Claude Invaders

A Space Invaders clone built with JavaScript.

## Setup

1. Install dependencies:
```
npm install
```

2. Build the game:
```
npm run build
```

3. Start the server:
```
npm start
```

4. Open your browser and navigate to `http://localhost:3000`

## Development

For development with hot-reloading:
```
npm run watch
```

## Controls

- Left/Right Arrow Keys: Move the player ship
- Spacebar: Shoot
